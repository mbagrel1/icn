Le système d'exploitation Windows


Cet OS a été crée par l'entreprise Microsoft, dont le fondateur est Bill Gates.
La première version de Windows est sortie le 20 novembre 1985, et elle
s'appuyait fortement sur le système d'exploitation MS DOS.
Aujourd'hui, la dernière version est Windows 10, depuis le 5 avril 2017.

Il y a eu de nombreuses versions de ce système d'exploitation comme Windows Vista, Windows XP, Windows 7, Windows10....Ce système est très connu puisqu'il est l'Os le plus utilisé chez les particuliers.

Pourquoi?

Car ce système est souvent fournis avec l'ordinateur et il existe en plusieurs versions. Il est aussi facile d'accès et très convivial. De plus la plupart des jeux sont développés pour fonctionner avec cet OS.
